# 유노알피지 (Youknow RPG)

판타지 세계관의 픽셀 아트 기반 선택형 RPG 게임입니다.  
분기 시스템과 멀티 엔딩, 스탯과 스킬 시스템이 포함되어 있습니다.

## 게임 링크  
[https://pion47.github.io/rpg-game/](https://pion47.github.io/rpg-game/)

## 주요 기능
- 캐릭터 스탯 및 스킬 포인트 시스템  
- 선택에 따른 분기 및 확률 적용  
- 10단계 스테이지, 5단계마다 특수 이벤트(상점, 보스)  
- 마나, 기력, 운 스탯과 확률 연동  
- 키보드 입력 기반 UI
